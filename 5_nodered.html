5. Node-RED

¿Qué es Node-RED?

Node-RED es una herramienta de desarrollo basada en flujo para programación visual para conectar dispositivos de hardware, API y servicios en línea como parte de la Internet de las cosas.

Se trabajará con Node-RED ya que puede ejecutarse en Raspberry Pi, que a su vez puede interactuar con Arduino para comunicarse con los dispositivos conectados al mismo (sensores, actuadores, cámaras, dispositivos de medida, etc.).

Esta aplicación tiene una interfaz muy visual, que simplifica el proceso de desarrollo del software, es decir, que a través de la programación por bloques permite configurar el programa de control del sistema de domótica que se diseñe
